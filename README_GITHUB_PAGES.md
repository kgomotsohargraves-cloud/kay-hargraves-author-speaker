# Kgomotso "Kay" Hargraves Author Website

This folder contains a static author and speaker website ready for GitHub Pages.

The site includes pages for Kay's author/speaker platform, books, media kit,
resources, booking, contact, book purchase links, and a book/speaking
newsletter. Current public book sections include Unmasking the Spirit of Fear
and Quieting the Storm.

## Files to upload

Upload everything inside this `website` folder:

- `index.html`
- `about.html`
- `books.html`
- `speaking.html`
- `media.html`
- `speaker-profile.html`
- `resources.html`
- `contact.html`
- `styles.css`
- `script.js`
- `assets/`

## GitHub Pages setup

1. Create a new GitHub repository.
2. Upload the files from this folder.
3. Go to repository `Settings`.
4. Go to `Pages`.
5. Under `Build and deployment`, choose `Deploy from a branch`.
6. Choose branch `main` and folder `/root`.
7. Save.

GitHub will give you a live website link.

## Before launch

The contact email is currently set to:

```js
const SITE_CONTACT_EMAIL = "connect@liberty-global-consulting.com";
```

Current live GitHub Pages website:

```text
https://kgomotsohargraves-cloud.github.io/kay-hargraves-author-speaker/
```

If you connect a custom domain later, use that domain everywhere instead.

Then update placeholder social links in `contact.html`.
